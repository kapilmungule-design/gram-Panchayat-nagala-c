Building Permission Details

Construction Approval

Before starting construction work, citizens need approval from Gram Panchayat.

  • ✔ Submit Application
  • ✔ Document Verification
  • ✔ Site Inspection
  • ✔ Permission Approval

Required Documents

  • 📄 Aadhaar Card
  • 📄 Property Ownership Document
  • 📄 Site Plan / Map
  • 📄 Passport Size Photo
  • 📄 Mobile Number
  • 📄 Previous Permission Copy (if available)
  • 📄 NOC Documents (if required)

Application Process

1. Submit Application

Fill permission form and submit required documents at Gram Panchayat office.

2. Document Verification

Officials verify submitted documents and details.

3. Site Inspection

Gram Panchayat team checks location if required.

4. Approval & Certificate

Approved permission certificate is issued.
